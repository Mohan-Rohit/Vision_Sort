# VisionSort – AI-Based Waste Classification System

# Project Description

> VisionSort is an AI-powered waste classification web application that automatically identifies waste materials from uploaded images using the YOLOv8 deep learning model. The system analyzes images and predicts the waste category along with a confidence score. It also provides appropriate disposal suggestions and maintains a history of previous predictions for user reference.

> This project demonstrates how artificial intelligence can be applied to support smart waste management and environmental sustainability.

# Features
1. AI-based waste image classification
2. Image upload functionality
3. YOLOv8 object detection model
4. Prediction confidence score display
5. Waste disposal suggestions
6. Prediction history tracking
7. Dashboard with basic statistics
8. Simple and user-friendly web interface
